php连接数据库

需要先下载

1.Phpstudy

2.navicat

步骤：

  1. 启动Apache和MySQL

  2. 打开网站，管理，修改，在高级设置中打开目录索引

  3. 在navicat Premium里新建MySQL连接，主机填要连接的主机ip，密码默认为root，然后测试连接，显示连接成功即可。

远程连接需要进行如下操作：

在此目录下cmd，

mysql -uroot -p然后输入密码root登录sql

然后进行以下操作

use mysql;
update user set Host=’%’ where User=’root’;
flush privileges;

就能远程连接其他主机的数据库了
