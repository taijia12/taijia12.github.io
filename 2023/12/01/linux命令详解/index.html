pwd 显示当前所在目录

cd 进入用户主目录；

cd ~ 进入用户主目录；

cd - 返回进入此目录之前所在的目录；

cd .. 返回上级目录(若当前目录为”/“，则执行完后还在”/“；”..”为上级目录的意思)；

cd ../.. 返回上两级目录；

cd !$ 把上个命令的参数作为cd参数使用

ls 查看当前目录下的文件，不能查看隐藏文件，默认为ls .

ls ..查看上一级目录下的文件

ls -a 查看所有文件，包括隐藏文件

ls -al 显示文件属性

ls /bin 查看bin目录下所有文件

ls /bin|grep mv 查找bin目录下所有包含mv的文件

ls /bin|more 查看 ，q停止查看

touch filename 创建文件

touch file1 file2 创建多个文件，将不同的文件名用空格分隔

touch+新文件+原文件 原文件创建时间覆盖新文件创建时间（新文件和原文件创建时间一样）

rm 删除

rm -f 强制删除

rm-r 递归删除

rm -rf 强制递归删除目录,将指定目录下的所有文件与子目录一并处理

rm -i 删除已有文件或目录之前先询问用户

mkdir [命令选项] dirname 创建文件夹

rmdir [命令选项] dirname 删除文件夹·

cp [命令选项] 源文件 目的文件 将一个或多个源文件或者目录复制到指定的目录中

-f：强行复制文件或目录，不论目标文件或目录是否已存在；
-i：覆盖既有文件之前先询问用户；
-p：保留源文件或目录的属性；
-R/r：递归处理，将指定目录下的所有文件与子目录一并处理。

mv [命令选项] 源文件 目标文件 对文件或目录重新命名，或者将文件从一个目录移到另一个目录中

-f：强行复制文件或目录，不论目标文件或目录是否已存在； 
-i：覆盖既有文件之前先询问用户；
-p：保留源文件或目录的属性；
-R/r：递归处理，将指定目录下的所有文件与子目录一并处理'。

mv newfile newdir/newfileCpy 将newfile剪切到newdir目录下，并重新命名为newfileCpy

usermod [命令参数] 参数

-c<备注>：修改用户帐号的备注文字；
-d<登入目录>：修改用户登入时的目录；
-e<有效期限>：修改帐号的有效期限；
-f<缓冲天数>：修改在密码过期后多少天即关闭该帐号；
-g<群组>：修改用户所属的群组；
-G<群组>；修改用户所属的附加群组；
-l<帐号名称>：修改用户帐号名称；
-L：锁定用户密码，使密码无效；
-s：修改用户登入后所使用的shell；
-u：修改用户ID；
-U:解除密码锁定。

查看文件内容：

cat 参数 文件名

-n 或 --number：由1开始对所有输出的行数编号；
-b 或 --number-nonblank：和-n相似，只不过对于空白行不编号。

head 参数 文件名

-n number：显示文件的前number行内容；

tail 参数 文件名

-n number：显示文件的最后number行内容；

ls 参数 目录

  • -a：显示所有档案及目录（ls内定将档案名或目录名称为“.”的视为影藏，不会列出）；
  • -A：显示除影藏文件“.”和“..”以外的所有文件列表；
  • -l：列出内容的详细信息；
  • -r：以文件名反序排列并输出目录内容列表；
  • -s：显示文件和目录的大小，以区块为单位；
  • -i：显示文件索引节点号（inode）。一个索引节点代表一个文件；
  • -R：递归处理，将指定目录下的所有文件及子目录一并处理。
  • -al: 查看目录下的所有信息(包括隐藏内容)，同时显示每个文件的详细信息

文件权限修改：

chmod ［who］ ［+ | - | =］ ［mode] 文件名

who参数如下：

u 表示“用户（user）”，即文件或目录的所有者；
g 表示“同组（group）用户”，即与文件属主有相同组ID的所有用户；
o 表示“其他（others）用户”；
a 表示“所有（all）用户”，它是系统默认值。

操作符如下：

+：添加某个权限；
-：取消某个权限；
=：赋予给定权限并取消其他所有权限。

mode参数如下：

r 可读；/4
w 可写；/2
x 可执行。/1

数字设定法具体命令如下：

mode为具体的数字。(r/w/x组合)

tar　-cvf　生成的tar包名　要打包的文件/目录列表

tar　-xvf　解压包名 -C 解压到指定目录
