linux命令详解

2023/12/1

ls 查看当前目录下的文件，不能查看隐藏文件，默认为ls .

ls ..查看上一级目录下的文件

ls -a 查看所有文件，包括隐藏文件

ls -al 显示文件属性

ls /bin 查看bin目录下所有文件

ls /bin|grep mv 查找bin目录下所有包含mv的文件

ls /bin|more 查看 ，q停止查看

touch+新文件+原文件 原文件创建时间覆盖新文件创建时间（新文件和原文件创建时间一样）

vi 编辑文件 i插入 Esc退出插入 :wq! 保存退出 :q! 不保存退出

cat -n +文件名 给文件加行数

cat

find / -name ls 在根目录下搜名字为ls的文件

​ -iname 区分大小写

​ *ls*搜含关键字ls的文件（*表示n个通配符）

​ *ls 以ls结尾

​ ls* 以ls开头

find / -size +204800 搜大于204800kb的文件

可以同时加多个限制

例如 find / -name ls -size +100 -type d

>> 追加

useradd admin 创建用户admin

password admin 设置密码

cd 切换路径

cd / 切换根目录

touch 创建文件，空格加文件名创建

d 目录文件

-普通文件

l 链接

​ 月 日 年

三个一组为权限

r:4/w:2/x:1 读/写/执行

mkdir test 创建目录

mkdir -p 创建文件夹，递归创建

cd 进home目录

目录详解看这

https://zhuanlan.zhihu.com/p/587321097

bin 包含应用程序的可执行文件（常用的二进制命令所在的目录，例如ls，cp，mkdir…）

boot Linux内核与系统引导程序（不建议对它操作）

dev目录 存储声卡，磁盘，光驱

etc yum ，rpm方式安装应用程序的默认配置文件路径

home包含用户的数据

usr包含用户安装的第三方应用 （有两个重要的子目录）

usr/local 编译方式安装程序的默认目录

usr/src 程序源码目录

var 系统与软件服务运行日志

lib 启动系统与运行命令所需的共享库文件与内核模块

proc 系统运行时，进程信息与内核信息存放辞目录

root Linux 超级用户目录，类似于Windows的administrator

sbin 系统管理命令存放目录，是超级用户root可执行命令的存放地

whoami 当前目录名字

pwd 当前目录

rm 删除

rm -f 强制删除

rm -rf 强制递归删除目录
